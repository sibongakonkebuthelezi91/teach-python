Learning notes

Story-style lessons with bright illustrations

Each note turns a Python idea into a simple game concept so children can remember it easily.

Variables are treasure boxes

Variables hold your game data so you can use it later. Think of them as labeled boxes for coins, lives, and scores.

coins = 5
print(coins)

Loops repeat your moves

Loops help your character repeat a jump, dance, or treasure check until the mission is complete.

for step in range(3):
    print("Jump!")

Conditionals are game rules

If a player has enough points, the game can say “You win!” If not, it can take another path.

if score > 10:
    print("You win!")

Functions are mini-games

Functions let you save a helpful action so you can play it again and again without rewriting it.

def collect_treasure():
    print("You found gold!")

Every new idea is a new level in your coding adventure.
