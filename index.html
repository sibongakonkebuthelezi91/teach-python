🧺 Variables = Treasure Chests

Variables hold your treasures so you can use them later. Think of them as labeled boxes. 

coins = 5
print(coins)

🔁 Loops = Repeating Moves

Loops let your character repeat an action until the mission is complete. 

for step in range(3):
    print("Jump!")

✅ Conditionals = Game Rules

If something is true, your game reacts. If not, it takes another path. 

if score > 10:
    print("You win!")

🛠️ Functions = Mini-Games

Functions let you save a useful action so you can play it anytime.

    print("You found gold!")
